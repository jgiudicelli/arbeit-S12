Arbeit Project - 2012 Version

This is the 2012 version of the Arbeit project using Rails 3. It was originally conceived as an excuse to play around with the awesome Chronic gem and requires this gem to run. It is currently being used to demonstrate some mulit-model forms as well as use of jQuery. It is a work in progress and additional features will be added over the next few weeks or so.

It does have its own populate file, so after migrating the database, you can simply run ‘rake db:populate’ to add fake (but somewhat realistic) data.

Getting the code

This project should be cloned using git (yes, you could get the zip file that github graciously provides, but don't -- it's easy so learn to clone projects using git). 

$ git clone git://github.com/profh/arbeit-S12.git

If you have already cloned the project and need to get an update, simply do the following:

$ cd your_local_directory/arbeit-S12/
$ git pull origin master
